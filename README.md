## üöÄ **Quick Start**

### **Prerequisites**

- **Node.js** >= 20.11.0 (Locked with `.nvmrc`)
- **npm** >= 10.8.0
- **Git** >= 2.30.0

### **1. Clone & Setup**

```bash
# Clone the repository
git clone https://github.com/your-org/ai-bos.git
cd ai-bos

# Install dependencies
npm install
```

### **2. Environment Setup**

```bash
# Copy environment files
cp railway-1/backend/env.example railway-1/backend/.env
cp railway-1/frontend/env.example railway-1/frontend/.env

# Edit with your configuration
nano railway-1/backend/.env
nano railway-1/frontend/.env
```

### **3. Manifestor Setup (Required)**

```bash
# Load all manifests and validate
npm run manifestor:load

# Check manifest health
npm run manifestor:health

# Validate all manifests
npm run manifestor:validate
```

### **4. Database Setup**

```bash
# Run database migrations
cd railway-1/backend
npm run migrate

# Seed initial data
npm run seed
```

### **5. Start Development**

```bash
# Start all services (monorepo)
npm run dev:all

# Or start individual services
cd railway-1/frontend && npm run dev  # Uses Turbopack by default
cd railway-1/backend && npm run dev
```

### **6. Access the Application**

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:3001
- **API Documentation**: http://localhost:3001/api/docs
